# Projekt: SCOOP – Source Code of the past

**Kurzname:** SCOOP  
**Kalender-Stichwort:** `SCOOP`

## Gegenstand

Creation of Ground Truth (GT) and accuracy measurement for historical source code analysis and recognition.

## Projektziele

- Entwicklung von Ground-Truth-Daten für historische Dokumente
- Accuracy-Messung für OCR/ICR-Systeme
- Evaluation und Benchmarking von Erkennungsalgorithmen

## Aufgaben

- Sammlung und Digitalisierung historischer Quellen
- Erstellung von Ground-Truth-Daten
- Entwicklung von Bewertungsmetriken
- Benchmarking verschiedener Erkennungssysteme
